# MuseumOfVisions – Digital Art Explorer

## Description
MuseumOfVisions is a web application designed to help users explore artworks from the Harvard Art Museums collection. The project demonstrates the use of JavaScript for API integration, data handling, and dynamic user interface development. It focuses on transforming raw data into an interactive and user-friendly browsing experience.



## Purpose
The purpose of this project is to:
- Understand how to fetch and display data from a public API
- Practice working with JavaScript, especially array methods such as filter, map, and sort
- Build a responsive and interactive user interface
- Develop problem-solving skills through real-world application development



## API Used
**Harvard Art Museums API**

Base URL:  
https://api.harvardartmuseums.org/object

This API provides access to a large collection of artworks, including details such as title, date, classification, and images.



## Planned Features

### Search
Users will be able to search for artworks based on keywords such as title. The application will display only the results that match the search query.

### Filtering
Users will be able to filter artworks based on specific categories such as classification (e.g., Paintings, Prints). This helps in narrowing down results.

### Sorting
- Date (chronological order)

### Pagination
The application will display a limited number of artworks per page and allow users to navigate between pages.

### Dark Mode
A toggle option will be provided to switch between light and dark themes for better user experience.



## Technologies Used
- HTML (structure of the web application)
- CSS (styling and layout)
- JavaScript (logic and interactivity)
- Fetch API (for making API requests)
